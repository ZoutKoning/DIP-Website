# Sem in Professional Issues II - Driver's Incentive Project
Developed by: <br />
 Jalen Barnes, James Barr, Jonathan Moore and Joseph Suter. <br />
This repo is to is to showcase the work we completed. <br />
Please do not copy or re-use the work contained in this repo. Thank you. <br />

For this project we had to preform full-stack development to create a website that would allow drivers to earn points from sponsors. <br />
We utilized Technologies: <br />
 * Database (Amazon RDS)
 * Live site (Amazon EC2)
 * Django
 * (IDE's): Pycharm, VS-Code, and MySQLWorkbench
